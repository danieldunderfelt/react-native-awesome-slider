## Overview

- Light & Dark theme
- Custom bubble & thumb
- Haptic mode
- Lottie thumb
- Track thumb mode

## Get Start

```sh
git clone https://github.com/alantoa/@danieldunderfelt/react-native-awesome-slider.git
cd @danieldunderfelt/react-native-awesome-slider
yarn bootstrap
```

### iOS

```sh
cd example
yarn ios
```

### Android

```sh
cd example
yarn android
```
